-Welcome 
-In this lab, we will build up fundamentals in number theory, specifically focusing 
on modular arithmetic and polynomials. Then, we will explore applications of 
these concepts in secret-sharing and error-correcting codes. 
-Introduction to the Math 
-Number Theory 
-Modular Arithmetic 
-Modular Arithmetic is a system in which numbers are limited to 
integers in a predefined range from 0 to N-1 
-Numbers that are larger or smaller than this range wrap around 
back into the range. 
-Consider a clock, which has N = 12:  
-15 hours after 1:00 is not 16:00, but instead 16-12 = 4:00. 
-4 hours before 1:00 is not -3:00, but instead -3+12 = 9:00. 
-Another way to think about it is that x (mod m) is the remainder 
when dividing x by m 
-x mod m = r -> x = mq + r for some q 
-Quick exercise: Write Python to convert an integer i potentially 
outside of the range with the provided N into the modular field 
-Inverses 
-One of the fantastic things about modular arithmetic is that it 
preserves most of the traditional arithmetic operations like add, 
subtract, and multiply 
-If x + y = r in normal arithmetic, then x + y = r (mod m) for 
any m as well. 
-The key difference comes in division. Another way of thinking 
about division is the multiplicative inverse 
-That is, x / y = x * (1/y), where 1/y is the multiplicative 
inverse of y 
-In modular arithmetic, we want to find the multiplicative inverse so 
that x * x^-1 = 1 (mod m) 
-Important: The multiplicative inverse for a number x mod m 
exists if and only if x and m are coprime - in other words, the 
greatest common divisor of x and m is 1. 
-So, to divide in modular arithmetic, simply multiply by the 
multiplicative inverse instead 
-How do we find the inverses? We can use an algorithm called 
Extended Euclids Algorithm. For now, we will provide this to you 
as a black box. Feel free to read through the code if you want, but 
its not required. 
-Galois Field 
-While it might seem weird or annoying that division functions 
differently in modular arithmetic, this weirdness is actually very 
useful in some applications 
-In normal arithmetic, dividing two integers sometimes creates a 
fractional answer. In modular arithmetic, multiplying by the 
multiplicative inverse always produces another integer in the 
range. Sweet! 
-A Galois Field (GF) allows us to use this property in multiple 
dimensions. A GF is essentially a modified version of the first 
quadrant of the Cartesian plane, where the horizontal and vertical 
axes are limited to the range [0, n-1] for GF(n) 
-Ex: write a function to graph the intersection of two lines with 
given slope and y-intercept in a Galois Field 
-Whoa! Its the same point as in normal math!
